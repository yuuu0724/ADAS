# 🚗 ADAS — 基于鲁班猫5 (RK3588) 的 YOLOv5 智能驾驶辅助系统

本项目在 **鲁班猫5 (RK3588)** 平台上部署 **YOLOv5**，实现了 **安全带检测** 与 **行人检测** 的实时识别系统。  
系统支持 **双模型并行推理**，能够从 **两个摄像头** 同时获取视频流，实时显示检测结果并进行 **语音播报提示**。

> 运行方法：直接运行 `ADAS_start/start/process` 主程序即可。

---

## 🧠 项目简介

本项目参考自：[YOLOv5_RK3588_object_detect](https://github.com/A7bert777/YOLOv5_RK3588_object_detect/tree/main)，  
原工程仅支持对 **单张图片的目标检测**。  

在此基础上进行了以下改进：

- ✅ **实时视频检测**：通过 OpenCV 读取摄像头画面，实现连续视频帧的目标识别。
- ✅ **双模型并行加载**：行人检测 + 安全带检测，实现同时识别多类目标。
- ✅ **多线程架构**：分别负责视频采集、推理、显示与音频播放，保证系统高效运行。
- ✅ **Qt 窗口显示双路视频**：通过自定义 `DisplayWindow` 类，将两路视频显示在不同窗口或屏幕区域。
- ✅ **语音播报提醒**：检测到“未系安全带”或“注意行人”时自动语音提示，提高驾驶安全性。
